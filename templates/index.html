API Documentation

POST /api/detect

Send a POST request with an image file to detect objects.

Request

curl -X POST -F "image=@/path/to/your/image.jpg" http://localhost:5000/api/detect

Response

{
  "success": true,
  "detections": [
    {
      "class_name": "Car",
      "confidence": 0.95,
      "bbox": [100, 200, 300, 400]
    },
    ...
  ]
}
